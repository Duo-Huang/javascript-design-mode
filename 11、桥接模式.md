## 介绍
桥接模式：桥接模式将抽象部分与实现部分分离，使它们都可以独立地变化，将继承关系转换为了关联关系，从而降低了类与类之间的耦合。

## 示例
```javascript
var class1 = function(a,b,c) {
    this.a = a;
    this.b = b;
    this.c = c;
};
var class2 = function(d) {
    this.d = d;
};
var demo = function(a,b,c,d) {
    this.one = new Class1(a,b,c);
    this.two = new Class2(d);
};
```
